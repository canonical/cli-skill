Annotations

Incomplete CRUD Sets

Engine domain:

  • Has show-engine (read) and use-engine (update/select)
  • Missing: no list-engines or engines command to discover available engines
  • Missing: no add-engine / remove-engine (engines are snap components)

Configuration domain:

  • Has get (read) and set (write)
  • Missing: no unset or reset command to restore defaults
  • Missing: no list-all-config command to show all current configuration

Verb Inconsistencies

  • use-engine uses "use" as a verb for selection — non-standard in Canonical vocabulary (DE013 favors enable/disable or set)
  • show-engine uses "show" correctly per DE013

Orphan Commands

  • chat — bare verb, implicit noun (acceptable for primary-type single-purpose tools per DE013)
  • completion — functions as a noun, not a verb; would be generate-completion in strict verb form
  • get / set — verbs without explicit nouns (standard pattern per DE013 for config access)
