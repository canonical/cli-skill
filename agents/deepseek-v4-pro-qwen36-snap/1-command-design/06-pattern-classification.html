Tradeoffs

Consolidation (Recs 1 & 2)

Pro: Reduces command count, eliminates confusion pairs, simplifies maintenance.
Con: Breaks existing workflows for debug command users (both hidden, impact minimal). Requires careful flag design.

Rename webui (Rec 3)

Pro: Complies with DE013 verb-noun standard, improves discoverability.
Con: Short memorable name lost. Backward compat requires alias for ≥1 major version. README must be updated.

Add deactivate-engine (Rec 4)

Pro: Completes engine lifecycle management.
Con: Adds command to already engine-heavy CLI. Use case may be narrow.

Table headers (Rec 5)

Pro: Aligns with DE013 and snap list conventions.
Con: Learned lowercase headers must be relearned. Any table-parsing scripts must update.
