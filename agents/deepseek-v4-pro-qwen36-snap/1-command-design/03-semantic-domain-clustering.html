Key Findings

  1. Over-fragmentation in Web UI: Three commands for one feature.
  2. Missing engine lifecycle commands: No add-engine or remove-engine.
  3. run is a domain anomaly: Engine-dependent but doesn't contain "engine" in name.
  4. Chat domain split: Two chat commands with different contracts should be consolidated.
  5. Imbalanced domains: Engine (6) and Web UI (3) dominate; most domains are singletons.
