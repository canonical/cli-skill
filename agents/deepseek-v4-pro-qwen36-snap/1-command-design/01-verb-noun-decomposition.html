Annotations

Incomplete CRUD Sets

  • config: Has get (read), set (create/update), unset (delete). CRUD complete ✅.
  • engine: Has list, show, use, validate, select. Missing: remove-engine (deactivation).
  • cache: Has prune (delete). Missing: read counterpart.
  • machine: Has show (read). Acceptable — machine info is read-only.
  • chat: Two chat commands in different namespaces. Inconsistent.
  • ui: Three web UI commands. Over-represented domain.

Verb Inconsistencies

  • use-engine uses use instead of select or activate.
  • prune-cache uses prune instead of remove or clean.
  • list-engines uses list prefix instead of bare plural engines per DE013.
