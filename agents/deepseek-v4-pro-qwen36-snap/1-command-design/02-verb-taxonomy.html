Key Findings

  1. Observation-heavy: 6 of 15 verbs (40%) are observation. Reflects management/monitoring role.
  2. No access verbs: No grant, revoke, enable, disable, login, logout. Root requirement is the only auth model.
  3. No transfer/migration verbs: All model management is via snap components.
  4. Punctual-dominant: Most verbs are instant state reads/writes. Long-running operations delegated to the server.
  5. webui misclassification: Noun used as command. Should be open-webui for verb-noun consistency.
  6. Debug namespace duplication: debug chat and debug serve-webui mirror top-level commands with different contracts.
