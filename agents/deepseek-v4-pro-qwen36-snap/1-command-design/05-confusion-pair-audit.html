Key Findings

  1. Debug namespace creates confusion (2 high-risk pairs): Debug variants mirror top-level commands with different semantics.
  2. list vs show convention gap: Standard per DE013 but unintuitive for novice users.
  3. webui vs serve-webui: Functionally complementary but names don't convey relationship.
  4. prune-cache and use-engine unlinked: Inverse-ish operations with no cross-reference.
  5. Overlapping information sources: get, show-engine, and status all surface similar info.
