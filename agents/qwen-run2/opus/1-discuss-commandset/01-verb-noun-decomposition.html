Verb Inconsistencies

IssueDetail
use vs setuse-engine selects the engine (a kind of "set"), but set is reserved for config keys. The verb use implies ongoing state rather than a point-in-time write.
show vs getshow-engine displays engine info (YAML dump), while get retrieves a single config value. Per DE013 §Commonly Used Commands, this is correct.
completion is a nounThe command uses a noun as the command name rather than a verb. Per DE013, commands should be verbs.
