Classification Notes

Observation: completion, get, and show reveal information rather than mutate state.

Mutation: set and use both change persisted runtime behavior.

Execution: chat and serve describe continuing activities, so they fit the atelic class.

Reversibility: the CLI has no perfect named inverse pairs. The clearest missing DE013 partner is unset for set.
