# Pattern Classification

| Command | Pattern | Notes |
|---|---|---|
| `get` / `set` | Key-Value Mutation | Standard configuration loop. |
| `show-engine` | State Inspection | Standard status query. |
| `use-engine` | State Mutation | Operates as a configuration switch, though written as an imperative verb. |
| `chat` | Interactive Session | Starts blocking REPL event loop. |
