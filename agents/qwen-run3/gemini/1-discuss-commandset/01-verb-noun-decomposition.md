# Verb-Noun Decomposition

| command | primary verb | primary noun | modifier/flag |
|---------|--------------|--------------|---------------|
| `use-engine` | `use` | `engine` | `--auto` |
| `show-engine` | `show` | `engine` | None |
| `get` | `get` | (Implicit Config) | None |
| `set` | `set` | (Implicit Config) | None |
| `chat` | `chat` | (Implicit Session) | None |
